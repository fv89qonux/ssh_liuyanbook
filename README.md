## 本项目实现的最终作用是基于SSH在线留言板系统
## 分为1个角色
### 第1个角色为用户角色，实现了如下功能：
 - QQ截图20201212093030
 - 修改头像和个人信息
 - 发表留言
 - 用户注册
 - 用户登录
